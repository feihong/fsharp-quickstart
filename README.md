# F# Quickstart

## Installation

As of July 2016, you can install Mono 4.2.1 using apt on Ubuntu 16.04. Latest version of Mono is 4.4.1.

```
apt install mono-complete fsharp
```

## Run script

```
fsharpi hello.fsx
```


## Compiling and running

```
fsharpc -o hello.exe hello.fs
mono hello.exe
```
